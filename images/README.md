# 图片资源目录

请将以下图片放入此目录：

## 必需的图片

### Hero 区域
- `hero-image.png` - 主要展示图片（建议尺寸：1024x768px）

### 功能图标
- `feature-1.svg` - 快速高效图标
- `feature-2.svg` - 简单易用图标
- `feature-3.svg` - 安全可靠图标
- `feature-4.svg` - 跨平台支持图标
- `feature-5.svg` - 智能同步图标
- `feature-6.svg` - 个性化定制图标

### 截图
在 `screenshots/` 目录下：
- `screenshot-1.png` - 应用主界面截图
- `screenshot-2.png` - 功能详情页面截图
- `screenshot-3.png` - 设置页面截图

### SEO 图片（可选）
- `og-image.jpg` - Open Graph 图片（建议尺寸：1200x630px）
- `twitter-image.jpg` - Twitter Card 图片（建议尺寸：1200x600px）

## 图片优化建议

1. 使用 WebP 格式以获得更好的压缩率
2. 为不同屏幕尺寸准备多个版本（使用 srcset）
3. 压缩图片以减小文件大小
4. 使用描述性的文件名

## 推荐工具

- **压缩工具**：TinyPNG, ImageOptim, Squoosh
- **格式转换**：CloudConvert, Convertio
- **图标资源**：Heroicons, Feather Icons, Font Awesome
